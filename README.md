# Проект Банк.

## Описание:

Проект Банк - это виджет, который показывает несколько последних успешных банковских операций клиента.

## Цель проекта:

Основой целью проекта является добавление новых фитч для личного кабинета клиента, 
которые на бэкенде будет готовить данные для отображения в новом виджете. Также данный проект
позволяет финансовым аналитикам быстро и удобно находить нужную информацию о транзакциях 
и проводить анализ данных.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Krimskiy-11/project_HOMEWORK_10.1.git
```

## Тестирование:

1. Тестирование всех функций в пакете tests
```
pytest tests/
```
2. Тестирование функций по отдельности:
```
pytest tests/test_reports.py
pytest tests/test_services.py
pytest tests/test_utils.py
pytest tests/test_views.py
```

## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Чтобы клонировать репозиторий с GitHub себе на компьютер, необходимо получить ссылку на доступ к репозиторию.
Найти ее можно в разделе Code на странице репозитория. Далее вам необходимо 
выбрать способ взаимодействия с GitHub (HTTPS) и скопировать ссылку на репозиторий для клонирования.
3. В главном меню PyCharm перейдите в меню Get from VCS
4. Вставьте ссылку на репозиторий и нажмите на кнопку Clone
5. По окончании клонирования будет открыт проект, вы можете работать с этим проектом локально.
